# Homebrew

**There is no Homebrew tap or `brew install` path for this project right now.** Install from npm:

```bash
npm i -g @burakboduroglu/portkill
# or
npx @burakboduroglu/portkill --help
```

Homebrew formulas do not use npm scoped names (`@burakboduroglu/portkill`); a future tap or [homebrew-core](https://github.com/Homebrew/homebrew-core) submission would be a separate decision.
